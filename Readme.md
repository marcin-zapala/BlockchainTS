# BlockchainTS
It will be complete solution to handle simple PoW based blockchain

(Work in progress)
## Setup

```
npm install --global lerna

git clone https://github.com/marcin-zapala/BlockchainTS.git

lerna bootstrap --hoist

lerna run tsc
```
